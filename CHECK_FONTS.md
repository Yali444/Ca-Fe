# בדיקת פונטים

אם הפונטים לא נטענים, בדוק:

1. **שמות הקבצים חייבים להיות בדיוק:**
   - `TimeBurner-Regular.woff`
   - `TimeBurner-Bold.woff`
   - `Yehuda-Regular.woff`
   - `Yehuda-Bold.woff`

2. **מיקום:** `public/fonts/` (לא `src/fonts/` או מקום אחר)

3. **אם הקבצים בשם אחר**, עדכן את `src/lib/fonts.ts` עם השמות הנכונים

4. **אחרי העלאת הקבצים:**
   - עצור את השרת (Ctrl+C)
   - הפעל מחדש: `npm run dev`
   - נקה את ה-cache אם צריך: `rm -rf .next` (או מחק את התיקייה `.next`)

5. **בדיקה בדפדפן:**
   - פתח DevTools (F12)
   - לך ל-Network tab
   - רענן את הדף
   - חפש קבצי `.woff` - הם צריכים להיטען

6. **אם עדיין לא עובד:**
   - בדוק את הקונסול לשגיאות
   - ודא שהקבצים לא פגומים
   - נסה להמיר ל-`.woff2` (יותר יעיל)
